Kontrollspørsmål:

"Forklar hvilken deler av din applikasjon så langt, implementerer de forskjellige lagene i MVC
arkitekturen. MVC står får Model­View­Controller, som til norsk kan oversettes til
Modell­Presentasjon­Kontorller."

	

	- Model: Modellen er HTML filen, siden det er denne filen som holder på dataen vi har. 
	- View:  Presentasjon er CSS filen, siden denne bestemmer hvordan vår data skal se ut.
	- Controller: Kontrolleren er JavaScript filen, som kan kontrollere hva som vises av informasjon.
