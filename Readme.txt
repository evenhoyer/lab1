0) Hva kan JavaScript brukes til?

	- JavaScript er et programmeringsspråk som kan brukes til mye i HTML-dokumenter.
	Noen eksempler på hva JavaScript kan gjøre:
		- Validering av det man skriver
		- Skjule eller vise ting på websiden
		- Endre HTML-bilder
		- Gjøre websider mer dynamiske (f.eks slipp og dra løsninger)
		
1) Hva er forskjellen på CSS id og class elementer?

	- CSS id er noe man bruker på objekter som kun blir brukt én gang på websiden.
	class elementer kan man bruke på objekter som går igjen flere ganger på websiden.

4) Hovedforskjeller mellom en <div> og <span> tag i HTML
 
	- Begge brukes hvis man ønsker å endre design på teksten som er imellom taggene. 
	Hovedforskjellen er at <div> tagen brukes for å designe en større mengde med tekst. Det kalles for et "block element".
	<span> brukes hvis man ønsker å endre en liten del med tekst, og kan også gjøre inni en  <div> tag. Det kalles for "innline element".
